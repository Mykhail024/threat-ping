# Threat Ping

Threat Ping is a cross-platform background application for real-time threat
notifications — air raid alerts, earthquakes, and other hazards.

The application monitors data from public alert APIs and delivers native OS
notifications without requiring a browser or manual monitoring.

The repository is in its initial state. Provider integrations, configuration
format, and documentation will be added incrementally as development progresses.
